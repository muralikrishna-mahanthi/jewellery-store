VALERION is a concept jewellery brand website designed to demonstrate:

  * Smooth page loading animation
  * Scroll-based reveal effects
  * 3D interactive background using Three.js
  * Clean luxury-inspired UI
  * Fully responsive layout

This project highlights frontend animation techniques and immersive web design.

Technologies Used:

  * HTML5
  * CSS3
  * Tailwind CSS (CDN)
  * JavaScript (Vanilla JS)
  * Three.js
  * GSAP
  * ScrollTrigger

  Note: JavaScript is written in store.html file

Future Improvements:

  * Add full eCommerce functionality
  * Connect backend for subscriptions
  * Improve mobile performance
  * Add product detail pages
